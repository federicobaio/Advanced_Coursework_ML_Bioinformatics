# Academic Coursework: Machine Learning & Bioinformatics

This repository is a collection of theoretical assignments, laboratory exercises, and research notes developed during my academic studies. It covers foundational and advanced topics in Machine Learning and Computational Biology.

## 🤖 Machine Learning
This section includes a series of homework assignments focusing on both the mathematical theory and practical implementation of ML algorithms:
* **Theories & Assignments:** Detailed solutions and derivations for ML concepts.
* **Implementation:** Python-based exercises covering core algorithms and data processing.

## 🧬 Bioinformatics
A deep dive into the computational methods used in biological research, with a focus on sequence analysis and protein structures:
* **Sequence Alignment:** Theoretical exercises on global and local alignment methods.
* **Structural Bioinformatics:** Analysis and study of GPCRs (G-Protein Coupled Receptors) and molecular modeling.
* **Topics covered:** Comprehensive assignments from Homework 1 to 5, covering the full curriculum of the course.

## 📂 Repository Structure
* `/Machine-Learning`: PDFs of theoretical assignments and Python source code.
* `/Bioinformatics`: A collection of 5 detailed reports on computational biology methods.
